Pada episode 12 ini mempelajari tentang pembuatan Custom Hook, Custom Hook sendiri sangat berguna jika kita akan
sering menggunakan data tersebut pada banyak komponen, karena dengan membuat custom hook kita dapat lebih mudah memanggil hook tersebut ketika saat dibutuhkan.

Custom Hook dapat berupa object maupun array. ketika berupa object maka nama function yang ada di dalam custom hook pada saat return tidak dapat dirubah,
ketika kita ingin memakainya di komponen tertentu. namun jika custom hook berupa array maka, kita dapat memberikan nama pada state kita yang dipanggil dari custom hook,
sesuai keinginan kita.
